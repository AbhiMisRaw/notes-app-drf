
# Notes App

This API is built using Django Rest Framework and Postgres DB

Clone the project

## How to run Locally

1. Select a directory
2. make an virtual environment
3. Clone the project
```bash
  git clone https://link-to-project
```

4. Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
